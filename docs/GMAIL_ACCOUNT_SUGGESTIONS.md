# Gmailアカウント名の候補

`information.bot@gmail.com`が既に使用されているため、以下の候補を提案します。

## 推奨候補

### 1. `info.delivery.bot@gmail.com` ⭐
- **理由**: 情報配信Botの目的が明確
- **長さ**: 適切
- **覚えやすさ**: 高い

### 2. `news.delivery.bot@gmail.com`
- **理由**: ニュース配信Botとして明確
- **長さ**: 適切
- **覚えやすさ**: 高い

### 3. `infobot.delivery@gmail.com`
- **理由**: シンプルで覚えやすい
- **長さ**: 適切
- **覚えやすさ**: 高い

### 4. `line.infobot@gmail.com`
- **理由**: LINE Botであることが明確
- **長さ**: 適切
- **覚えやすさ**: 高い

### 5. `info.collector.bot@gmail.com`
- **理由**: 情報収集Botとして明確
- **長さ**: やや長い
- **覚えやすさ**: 中

## 命名規則の提案

以下のパターンで候補を生成できます：

1. **目的 + delivery + bot**
   - `info.delivery.bot@gmail.com`
   - `news.delivery.bot@gmail.com`
   - `data.delivery.bot@gmail.com`

2. **目的 + bot + delivery**
   - `infobot.delivery@gmail.com`
   - `newsbot.delivery@gmail.com`

3. **line + 目的 + bot**
   - `line.infobot@gmail.com`
   - `line.newsbot@gmail.com`

4. **目的 + collector + bot**
   - `info.collector.bot@gmail.com`
   - `news.collector.bot@gmail.com`

## 選択時の注意点

- ドット（`.`）の位置は自由に変更可能（Gmailでは無視される）
- プラスエイリアス（`+`）は使用可能
- アンダースコア（`_`）は使用不可
- ハイフン（`-`）は使用可能だが、推奨されない

## 推奨

**最優先候補: `info.delivery.bot@gmail.com`**

理由：
- 目的が明確（情報配信Bot）
- 覚えやすい
- プロフェッショナルな印象
- プラスエイリアスで複数サイトに対応可能

**使用例:**
- 基本アドレス: `info.delivery.bot@gmail.com`
- AI Weekly用: `info.delivery.bot+aiweekly@gmail.com`
- ドローン用: `info.delivery.bot+drone@gmail.com`
